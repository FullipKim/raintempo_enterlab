ING Studio Website

360VR Museum Website

360vr museum platform website design.


메타버스 기반 뮤지엄 증강현실 관람 플랫폼 360vr뮤지엄의 웹사이트를 리디자인했습니다.
메타포트를 활용한 기술을 특징으로 삼고 있기에, 메타포트 내 미니맵 3D공간을 메인비주얼에 표현하여 아이덴티티를 살렸고, 브랜드 컬러인 미드나잇 바이올렛 컬러를 그대로 활용하여 디자인했습니다.
미드나잇 바이올린은 묵직하고 진중한 느낌을 주는 컬러로, 박물관 관람 시 느낄 수 있는 조도가 낮은 환경을 연상시키는 효과가 있어서 웹사이트에서도 컨텐츠에 집중할 수 있는 환경을 만들어 주었습니다. 하단 컨텐츠 부분에는 백그라운드 컬러를 명도가 높은 그레이컬러를 사용하여 콘텐츠들의 다양한 색감을 차분하게 눌러주었습니다.

DD entertainment Website

DD entertainment co.,Ltd Web design and Publishing


이벤트와 대형 기업행사, 국내 공공행사를 주로 운영대행하는 주식회사 디디엔터테인먼트입니다.
차분한 듯 생기가 넘치는 미들톤 블루컬러를 제안주셨고, 로고 리디자인을 통해 다양한 색 조합 베리에이션이 가능하여 재미있는 작업이였습니다.
때로는 로고의 메인 브랜드컬러인 블루 투톤을 배경으로 사용하기도 하고,
깔끔하고 차분한 인상을 주고싶은 섹션에는 블루톤이 살짝 가미된 밝은 화이트톤 그레이 컬러를 배경으로 사용해주었습니다.
클라이언트 측의 고민은 브랜드 아이덴티티를 재정비하고 마케팅에 활용하고자 하는 것이였기에,
전체적인 톤앤매너와 디자인가이드를 잡아드렸습니다.

Dessert Brand Cafe Il-an Website

Cafe Il-an Brand Site


쌀로 만든 건강한 글루텐프리 디저트를 직접 만들고 판매하는 브랜드 카페일안의 홈페이지 시안입니다.
전체적으로 실제 매장을 직접 가본 뒤, 매장 내부의 분위기에 맞게 가이드를 잡고 작업하였습니다.
카페일안의 메인 톤은 딥그린이지만, 이전 매장의 아지트같은 느낌보다 새로 확장이전한 매장만의 깔끔하고 탁 트인 사랑스러운 공간의 무드를 담아내고자 뉴트럴톤을 주로 사용해주었습니다.
브랜드 마케팅 콘텐츠들을 살펴보니 레트로 하이틴 무드를 약간 가미한 뉴트로 디자인을 즐겨 쓰고 계셨습니다.
그러면서도 깔끔하고 차분한 느낌을 놓지 않아 전체적으로 정갈하다는 느낌을 받아 디자인 시 심플하고 미니멀한 특징을 살려보았습니다.

360xcon Admin dashboard

360xcon Admin dashboard Publishing


B2B 메타버스 솔루션 플랫폼 360엑스콘의 쇼룸 솔루션 어드민 대시보드입니다.
360도로 돌려 볼 수 있는 기업형 가상 온라인 쇼룸을 편리하게 편집하고, 통계지표를 관리할 수 있는 관리자화면을 퍼블리싱하였습니다.
총 3 가지의 어드민을 갖추고 있었고, 각 솔루션 별로 정해진 심볼과 메인 컬러가 있어서 쇼룸의 메인컬러인 미들톤 민트를 활용하여 작업했습니다.
깔끔하면서도 한 눈에 모든 기능이 들어올 수 있는 관리자 화면권한을 고려하였고,
사용자가 편리하게 이용 할 수 있도록 아이콘과 텍스트 병행표기, 드롭다운 메뉴구성을 중점적으로 UX에 반영한 UI입니다.

D-Vent Wordpress Website

Business Event Company Website UI Design & Publishing


디벤트는 기존 견적문의를 통한 터무니없는 바가지요금을 당연시하던 국내 기업이벤트시장의 단점을 파악하고 개선하고자하는 새로운 솔루션입니다.
쇼핑몰같은 형태로 이루어진 커뮤니티형 서비스 플랫폼으로, 워드프레스를 기반으로 원하는 키워드를 한번에 쉽게 검색할 수 있습니다.
검색필터 기능과 함께 메인 화면의 베스트셀러 진열 섹션, 그리고 전체적인 느낌을 브랜드 핵심가치에 초점을 맞춰 디자인하고 UX 플로우를 기획하여 퍼블리싱했습니다.
